# Beauracracy Dashboard

A SaaS-style dashboard for complaint triage using AI-based risk scoring.

## 🚀 Features
- Risk-based prioritization (HIGH / MEDIUM / LOW)
- AI Suggested Actions
- Complaint queue system
- Dashboard analytics
- FastAPI backend + React frontend

## 🛠️ Tech Stack
- React + Tailwind CSS
- FastAPI
- Recharts
- JWT Auth (optional)

## ▶️ Run Locally

### Backend
uvicorn main:app --reload

### Frontend
npm install
npm run dev
